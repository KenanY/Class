Class.js

simple, elegant inheritance for your JavaScript
GitHub Download
  • Allows inheritance through a convenient .extend() method.
  • Plays nicely with AMD (require.js for example) or CommonJS.
  • Works either in the browser (one Class global variable) or in a Node environment through module.exports.
  • File size: Less than 1 kb

Usage

If you'd like to use Class.js in a Node environment, you can install it using [npm](https://npmjs.org/package/node-classjs):

`npm install node-classjs` `var Class = require('node-classjs');`
